## ⚠️ 触发:用户真机验 11 张截图反馈 3 个问题

### 问题 1:日期选择器英文(iOS)
- **截图 244**:账户编辑弹层点日期 → `Select date / Thu, Jan 1 / January 2026 / Cancel / OK`
- **根因**:main.dart 没配 localizations,showDatePicker 拿不到 zh_CN
- **修**:pubspec 加 `flutter_localizations` + main.dart 加 delegates + 3 个 `_pickDate` 显式 `locale: Locale('zh','CN')`

### 问题 2:借出/借入应是独立记账界面
- **截图 261-264**:咔皮的「借出」「借入」是 **独立全屏页面**,不是账户弹层预选借贷
- 关键差异:
  - 顶部黄框 `起始余额/起始欠款`
  - `起始时间(必填)` + 提示「该时间之前的记录不计入余额统计」
  - **`扣款账户`(借出) / `入款账户`(借入)** —— 真正的双账户联动
  - 3 toggle:计入净资产 / 设为特别关注账户 / 设为默认收账/支出
- **根因**:我之前把「+」菜单借出/入路由到 `AccountEditSheet(initialSubType:...)`,语义错位
- **修**:
  - schema v6→v7 migration:transactions 加 `fromAccountId/toAccountId/counterpartyName/startDate`
  - TransactionType 加 `lend/borrow`
  - DAO 新增 `lendMoney` / `borrowMoney`(双账户事务 + 借贷分类自动 seed)
  - 新建 `LendRecordPage` / `BorrowRecordPage` 全屏独立 widget
  - 主页「+」菜单路由修正到新页面

### 问题 3:转账下拉只有现金
- **截图 244**:用户 6 个账户,转账扣款下拉只显示「现金」
- **根因**:`transferableAccountListProvider` 硬编码只过滤 `fund + recharge`
- **修**:去掉过滤,返回全部账户;DAO 通用处理(toAccountId 必须存在即可)
