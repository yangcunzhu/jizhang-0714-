### 4. 关键技术细节

1. **autoDispose 时序坑修复**:ActionSheet pop → showRecordSheet 期间 `recordFormProvider` 被 dispose,`loadForEdit` 状态丢失。改为 `showRecordSheet(context, editing: tx)`,由新 sheet 在 `initState` 用 `addPostFrameCallback` 调 `loadForEdit`(此时 recordFormProvider 已被 watch)。
2. **`submit()` 编辑/新建双分支**:根据 `state.editingTransactionId` 走 UPDATE(保留原 occurredAt / createdAt)或 INSERT,提交后 vibrate 档位不同(heavy vs light)。
3. **退款 type 反向 + 分类自动选第一个反向类**:Stage 1 没有"退款"专用分类,自动选第一个反向 type 分类(支出 → 工资)。Stage 2 自定义分类时,用户可指定退款目标分类(待 S02 决策)。
4. **note 智能拼接**:`退款` + ` · ` + 原 note(若非空);空 note 时仅 "退款",避免 "退款 · " 丑尾巴。
5. **`Haptics` private constructor**:`Haptics._()` 禁实例化,纯静态工具类。
6. **SnackBar 文案区分**:`已记账`(新建)vs `已修改`(编辑)vs `已退款`(退款)vs `已删除`(删除),4 个 action 4 种文案 — 用户一眼分清操作结果。
