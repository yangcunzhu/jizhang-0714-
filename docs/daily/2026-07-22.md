## 🐛 预计问题

| 问题 | 解决 |
|---|---|
| 长按 ActionSheet 默认 500ms 触发 | `LongPressDraggable` 或 `GestureDetector onLongPress` 默认 500ms,够 |
| TransactionTile.onLongPress 通知父组件 | 父组件回调参数 — 走 ConsumerWidget + 弹 Dialog |
| 修改交易:复用 recordSheet? or 新 edit 页? | 复用 recordSheet,initialNote/initialAmount/initialCategoryId 从 transaction 反向填充 |
| 退款 flag 类型:bool vs enum? | enum RecordMode { create, edit, refund } 灵活 |
| `vibration.vibrate(duration: 100)` 等待 | 现有 fire-and-forget 包装,统一加长振 |
| Key 命名规范:`Key('semantic-name')` | ADR-0014 已定义语义风格 |
