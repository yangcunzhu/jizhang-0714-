| 类型 | 产品定位 | 还款策略(本次)|
|---|---|---|
| 💵 现金 | 物理现金 | 无 |
| 🏦 储蓄 | 银行存款 | 无 |
| 💳 信用卡 | 信用卡 | 储蓄/现金 → 信用卡 ✅ |
| 🅰️ 花呗 | 蚂蚁花呗 | 储蓄/现金 → 花呗 — **本次补** |
| 🆘 网贷 | 借呗/微粒贷等 | 储蓄/现金 → 网贷 + **期数管理** — **本次补** |

### 3. schema v5 migration(ADR-0026 §实施清单第 2 项)

- `transactions` 表加 `installmentPeriod INTEGER`(nullable,网贷期数)
- 沿用 ADR-0017 模式:`m.addColumn(transactions, transactions.installmentPeriod)`
- 现有数据自动为 null(无需 backfill)

### 4. transferRepayment 接口扩展(ADR-0026 §1 + §3)

- 参数 rename:`fromSavingsAccountId` → `fromAccountId`,`toCreditCardAccountId` → `toAccountId`
- 新增 `installmentPeriod`(网贷专属,可空)
- 校验升级(3 类场景全覆盖):
  - **正常**:储蓄够 + 收款账户合法 → 成功
  - **异常(余额不足)**:抛 StateError,事务回滚
  - **边界(amount=0)**:抛 ArgumentError
  - **网贷还款必须传 installmentPeriod >= 1**

### 5. 还款弹层加期数下拉(ADR-0026 §3)

- 新建 `_InstallmentPeriodPicker` widget
- 6/12/24/36 期 ChoiceChip 选项
- 仅网贷还款时显示(type == onlineLoan)

### 6. 主页「距离还款日 X 天」提醒卡片(ADR-0026 §1)

- 新建 `_RepaymentReminderCard` widget,在净资产占位卡之上
- 列出欠款类账户(信用卡/花呗/网贷),按 dueDay 排序,显示最近的 1 个
- urgency 颜色:≤3 天红 / ≤7 天橙 / >7 天灰
- 文字:「今天」「明天」「X 天后」+ emoji + 账户名
- 过滤:过期(< 0)或 > 30 天不显示

### 7. transaction_tile 显示「12 期」徽章(ADR-0026 §1)

- `transaction.installmentPeriod != null` → 紫色徽章
- subtitle 改 Row(text + 徽章),Flexible 防止溢出
