### 关键技术决策
1. **databaseProvider 用 `overrideWithValue` 模式**:测试 setUp 创建 db,tearDown close;ProviderContainer 接管 stream 生命周期,避免 `ref.onDispose` 链在 fake_async 中悬挂 timer。
2. **Widget 测试用 `UncontrolledProviderScope` + 预先 `.future`**:同步等 stream 首事件发出后 pumpWidget,绕开 Drift watch 真 timer 与 fake_async 冲突。
3. **测试找"午饭"用 `find.textContaining`**:`TransactionTile.subtitle` 把 note 拼到 `"MM-dd HH:mm · 午饭"`,完整 `find.text` 不命中,改 substring 匹配。
4. **金额分→元静态格式化**:`cents ~/ 100` + `padLeft(2,'0')`,暂不引 `NumberFormat`(Stage 5 仪表盘引入 fl_chart 时再加)。
