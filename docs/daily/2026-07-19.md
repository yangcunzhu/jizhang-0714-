## 🐛 预计问题

| 问题 | 解决 |
|---|---|
| Widget 测试拿不到真实 DB | ProviderScope `overrides:` 注入 `AppDatabase.forTesting(NativeDatabase.memory())` |
| 金额分→元显示 | intl `NumberFormat`,`amountCents/100`,保留 2 位 |
| StreamProvider 初始 loading 态 | `.when(data/loading/error)` 处理三态,空态友好提示 |
| databaseProvider 泄漏 | `ref.onDispose(db.close)` |
