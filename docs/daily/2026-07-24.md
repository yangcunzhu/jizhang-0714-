## 🔴 真机手验 3 个核心场景

> 等同 `integration_test/e2e_*.dart` 的真机手动跑(S01 ROA 必经)

### 场景 1:记账主流程

```
1. iPhone 启动 App(爱思助手装 Runner.ipa)
2. 看主页(空态:净资产 + 提示 + FAB)
3. 点"记一笔"FAB
4. 选分类"餐饮"🍔
5. 输入金额 1 2 . 3 4
6. 点"下一步"
7. Step 3:看到账户"现金"💵 + "添加账户"占位按钮 + 备注输入框
8. 备注输入"午饭"(可选)
9. 点"保存"
10. 看到主页"攒攒 +1 ✨"动效 1.5s
11. 主页列表显示"🍔 餐饮 午饭 ¥12.34"
12. 振动反馈(50ms 短振)
```

### 场景 2:真持久化

```
1. 上一步主页列表显示 1 笔交易
2. iPhone 锁屏(右上滑)
3. 等 5s
4. 解锁回 App
5. 主页仍显示"🍔 餐饮 午饭 ¥12.34"(数据未丢)
```

### 场景 3:修改 + 退款 + 删除

```
1. 长按主页交易"🍔 餐饮 午饭 ¥12.34"
2. 弹出 ActionSheet(编辑 / 退款 / 删除)
3. 点"退款"
4. recordSheet 复用,显示"🍔 餐饮"作为默认分类,账户默认"现金"
5. 不动直接保存(测试反向流程)
6. 主页列表多一笔反向交易(收入方向 +¥12.34)
```
