## 🆕 Day 16 关键实现

### 1. widget test 卡死根因 + 修复

**根因**:`CategoryTemplatePage` 用 `StreamProvider<List<CategoryTemplateEntry>>`(在 `category_template_provider.dart`),Drift `watchAllTemplates()` 返回真 stream。`flutter_test` 默认 fake_async 时钟,Drift stream 的真 timer 永远不会触发,widget tree 一直等待 `templatesAsync.when(...)`,导致 `flutter_tester` 进程挂起。

**修复**:照搬 `test/features/home/home_page_test.dart` 的 bootContainer 模式:

```dart
Future<ProviderContainer> bootContainer(WidgetTester tester) async {
  final container = ProviderContainer(
    overrides: [databaseProvider.overrideWithValue(db)],
  );
  addTearDown(container.dispose);
  await container.read(templateListProvider.future);  // ← 关键:.future 同步拿 stream 首个 event
  return container;
}
```

**结果**:7/7 widget 测试全过(<2s),不再卡死。沿用模式对其他 S02 page(用 StreamProvider 的)同样有效。
