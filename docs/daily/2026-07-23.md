### 4. 关键技术细节

1. **OverlayEntry 自管理生命周期**:ConfettiBurst fire 时插入 OverlayEntry,500ms 动画完成后 onComplete → entry.remove()。无需在外部维护 List<OverlayEntry>,零泄漏。
2. **GlobalKey 而非 LocalKey**:`recordFabKey` 是 GlobalKey,可在 widget tree 外部 `currentContext.findRenderObject()` 拿屏幕坐标。LocalKey 找不到 currentContext,这是 Day 9 调试时发现的细节。
3. **BuildContext across async gaps**:ActionSheet onTap 是 async,`notifier.submitAsRefund()` 后 context 可能已 mounted=false。已加 `if (context.mounted)` 守卫,过 analyzer。
4. **showRecordSheet 返回 Future<bool>**:用 `.then((result) => result ?? false)` 把 nullable bool 转 non-null,默认 false(用户主动关闭)。
5. **E2E workflow 自动 boot 模拟器**:用 `grep -oE "[0-9A-F]{8}-..." | head -1` 抓 UDID,适配任意 Xcode 版本(iPhone 15/16/17...)。
6. **flutter analyze + flutter test 在 E2E workflow 内双重跑**:unit + widget 先挡住明显 bug,避免无谓启动 iOS Simulator,省 CI 分钟。
