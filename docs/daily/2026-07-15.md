**09:30-10:30 安装 Flutter SDK（Windows）**

官方下载：https://docs.flutter.dev/get-started/install/windows

- [ ] 下载 Flutter SDK 稳定版（zip 格式，约 700MB）
  - 推荐：清华镜像 https://mirrors.tuna.tsinghua.edu.cn/flutter/flutter_infra_release/releases/
  - 选择 `flutter_windows_3.24.5-stable.zip`（或更新）
- [ ] 解压到 `C:\src\flutter`（**不要**解压到 `C:\Program Files\`）
- [ ] 配置 PATH：系统属性 → 高级 → 环境变量 → 用户变量 Path → 添加 `C:\src\flutter\bin`
- [ ] 新开 PowerShell 验证：
  ```powershell
  flutter --version
  # 预期：Flutter 3.24.x · Dart 3.5.x · DevTools 2.x
  ```
- [ ] 配置国内镜像（加速后续包下载）：
  ```powershell
  $env:PUB_HOSTED_URL = "https://pub.flutter-io.cn"
  $env:FLUTTER_STORAGE_BASE_URL = "https://storage.flutter-io.cn"
  # 永久配置（写入 PowerShell profile）：
  notepad $PROFILE
  # 添加以上两行
  ```

**10:30-11:30 安装 VS Code + 插件**

- [ ] 下载 VS Code：https://code.visualstudio.com/Download
- [ ] 安装（全部默认选项，勾选"添加到 PATH"）
- [ ] 打开 VS Code → 左侧 Extensions → 搜索并安装：
  - [ ] **Flutter**（官方插件，会自动装 Dart）
  - [ ] **Dart**（如果没自动装）
  - [ ] **Error Lens**（强烈推荐，错误高亮）
  - [ ] **GitLens**（Git 增强）
  - [ ] **Todo Tree**（TODO 标记）
- [ ] 安装中文语言包（可选）：搜索 `Chinese (Simplified)` → Install

**11:30-12:00 配置 Git**

- [ ] 打开 PowerShell：
  ```powershell
  git config --global user.name "Your Name"
  git config --global user.email "your.email@example.com"
  git config --global init.defaultBranch main
  git config --global core.autocrlf false   # Windows 防 LF/CRLF 问题
  git config --global core.editor "code --wait"
  ```
- [ ] 验证：
  ```powershell
  git config --list | Select-String "user"
  ```
- [ ] 生成 SSH key（GitHub 推荐方式）：
  ```powershell
  ssh-keygen -t ed25519 -C "your.email@example.com"
  # 一路回车，不设密码
  cat ~/.ssh/id_ed25519.pub
  # 复制输出的公钥，备用（Day 2 配 GitHub 用）
  ```

**19:00-20:30 GitHub Actions workflow 草稿**

- [ ] 创建目录：
  ```powershell
  New-Item -ItemType Directory -Force .github\workflows
  ```
- [ ] 创建 `.github/workflows/build-ios.yml`：
  ```yaml
  name: Build iOS .ipa
  
  on:
    push:
      branches: [ main ]
    workflow_dispatch:  # 允许手动触发
  
  jobs:
    build:
      runs-on: macos-14  # GitHub Actions macOS runner
      
      steps:
        - name: Checkout
          uses: actions/checkout@v4
        
        - name: Setup Flutter
          uses: subosito/flutter-action@v2
          with:
            flutter-version: '3.24.5'
            channel: stable
        
        - name: Install dependencies
          run: flutter pub get
        
        - name: Generate certificates (TODO: Day 2 配置)
          run: |
            echo "TODO: 证书配置将在 Day 2 完成"
        
        - name: Build .ipa (without signing for now)
          run: |
            flutter build ios --release --no-codesign
        
        - name: Upload artifact
          uses: actions/upload-artifact@v4
          with:
            name: jizhang-app-unsigned
            path: build/ios/iphoneos/Runner.app
            retention-days: 7
  ```
- [ ] **注意**：Day 1 只是草稿，真正的签名 build 在 Day 2 配置
