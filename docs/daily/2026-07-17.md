#### 详细步骤

**09:00-10:00 旧电脑准备**

⚠️ **旧电脑要求**（用户提前准备）：
- macOS 10.14+ 或 Windows 10+（Mac 更稳定）
- iTunes 已安装（Windows 必须）
- 能一直开着（或设置成唤醒不休眠）
- 与 iPhone 在同一 WiFi（首次配对可以 USB）

- [ ] 打开旧电脑，确认网络通畅
- [ ] 浏览器打开 AltStore 官网：https://altstore.io/
- [ ] 下载 AltServer：
  - macOS：`AltServer.dmg`
  - Windows：`AltInstaller.exe`

**10:00-11:00 AltServer 安装**

- [ ] **macOS**：
  - [ ] 双击 `AltServer.dmg` → 拖到 Applications
  - [ ] 打开 Launchpad → 点击 AltServer
  - [ ] 菜单栏出现 AltServer 图标（🌐）
  - [ ] 点击图标 → `Install Mail Plug-in`（用于自动续签邮件）
  - [ ] 打开 Mail → Mail → 设置 → 插件管理 → 启用 AltPlugin
- [ ] **Windows**：
  - [ ] 右键 `AltInstaller.exe` → 以管理员身份运行
  - [ ] 安装 iTunes + Apple Mobile Device Support（如未装）
  - [ ] 启动 AltServer（任务栏图标）
- [ ] **Apple ID 配置**：
  - [ ] 点击 AltServer 菜单 → `Sign in with Apple ID`
  - [ ] 输入 Apple ID + 密码（**使用 App-Specific Password**，不要用主密码）
    - 生成方式：https://appleid.apple.com → App-Specific Passwords → 生成
    - 标签：`AltStore-Auditor`
  - [ ] 验证：菜单显示 Apple ID 邮箱

**11:00-12:00 旧电脑常开设置**

- [ ] **macOS**：
  - [ ] 系统设置 → 节能 → 防止显示器关闭时电脑进入睡眠：`永不`
  - [ ] 系统设置 → 锁定屏幕 → 关闭 `屏幕保护启动后立即`
  - [ ] 共享 → 远程管理 → 允许（方便远程）
- [ ] **Windows**：
  - [ ] 控制面板 → 电源选项 → 更改计算机睡眠时间 → `从不`
  - [ ] 关闭屏幕保护
- [ ] 网络：
  - [ ] 旧电脑 + iPhone 都在同一 WiFi
  - [ ] 旧电脑 IP 地址记录（备用）：`ipconfig` / `ifconfig`
- [ ] 防火墙：
  - [ ] 允许 AltServer 通过防火墙（首次启动会弹窗）
