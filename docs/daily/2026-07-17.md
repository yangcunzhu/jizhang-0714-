#### 详细步骤

**09:00-10:00 旧电脑准备**

⚠️ **旧电脑要求**（用户提前准备）：
- macOS 10.14+ 或 Windows 10+（Mac 更稳定）
- iTunes 已安装（Windows 必须）
- 能一直开着（或设置成唤醒不休眠）
- 与 iPhone 在同一 WiFi（首次配对可以 USB）

- [ ] 打开旧电脑，确认网络通畅
- [ ] 浏览器打开 AltStore 官网：https://altstore.io/
- [ ] 下载 AltServer：
  - macOS：`AltServer.dmg`
  - Windows：`AltInstaller.exe`

**10:00-11:00 AltServer 安装**

- [ ] **macOS**：
  - [ ] 双击 `AltServer.dmg` → 拖到 Applications
  - [ ] 打开 Launchpad → 点击 AltServer
  - [ ] 菜单栏出现 AltServer 图标（🌐）
  - [ ] 点击图标 → `Install Mail Plug-in`（用于自动续签邮件）
  - [ ] 打开 Mail → Mail → 设置 → 插件管理 → 启用 AltPlugin
- [ ] **Windows**：
  - [ ] 右键 `AltInstaller.exe` → 以管理员身份运行
  - [ ] 安装 iTunes + Apple Mobile Device Support（如未装）
  - [ ] 启动 AltServer（任务栏图标）
- [ ] **Apple ID 配置**：
  - [ ] 点击 AltServer 菜单 → `Sign in with Apple ID`
  - [ ] 输入 Apple ID + 密码（**使用 App-Specific Password**，不要用主密码）
    - 生成方式：https://appleid.apple.com → App-Specific Passwords → 生成
    - 标签：`AltStore-Auditor`
  - [ ] 验证：菜单显示 Apple ID 邮箱

**11:00-12:00 旧电脑常开设置**

- [ ] **macOS**：
  - [ ] 系统设置 → 节能 → 防止显示器关闭时电脑进入睡眠：`永不`
  - [ ] 系统设置 → 锁定屏幕 → 关闭 `屏幕保护启动后立即`
  - [ ] 共享 → 远程管理 → 允许（方便远程）
- [ ] **Windows**：
  - [ ] 控制面板 → 电源选项 → 更改计算机睡眠时间 → `从不`
  - [ ] 关闭屏幕保护
- [ ] 网络：
  - [ ] 旧电脑 + iPhone 都在同一 WiFi
  - [ ] 旧电脑 IP 地址记录（备用）：`ipconfig` / `ifconfig`
- [ ] 防火墙：
  - [ ] 允许 AltServer 通过防火墙（首次启动会弹窗）

#### 🚧 上午阻塞 / 风险

- ⚠️ macOS Gatekeeper 阻止：右键 → 打开 → 仍要打开
- ⚠️ Apple ID 登录失败：用 App-Specific Password
- ⚠️ 旧电脑休眠：电源设置调整

- [ ] 输入 Apple ID + 密码（App-Specific Password）
- [ ] AltStore 自动安装到 iPhone

方法 B：手动下载安装（如果方法 A 失败）

- [ ] 点击 iOS 下载链接
- [ ] 设置 → 通用 → VPN 与设备管理 → 信任证书

**14:00-15:00 AltStore 配对**

- [ ] 打开 iPhone 上的 AltStore
- [ ] `Settings` tab → `Sign in with Apple ID`
- [ ] 输入 Apple ID + App-Specific Password
- [ ] 验证成功
- [ ] `Settings` tab → `WiFi Sync` → 启用
- [ ] AltStore 会显示连接的电脑（需旧电脑 AltServer 也在运行）

**15:00-16:00 安装 .ipa**

- [ ] 在 GitHub Actions 下载 Day 2 产出的 `Runner.ipa`：
  - 浏览器打开 https://github.com/YOUR_USER/jizhang-app/actions
  - 找到 Day 2 成功的那次 workflow
  - 底部 Artifacts → 下载 `jizhang-app-signed.zip`
  - 解压得到 `Runner.ipa`
- [ ] 把 `Runner.ipa` 通过以下方式传到 iPhone：
  - **AirDrop**（最简单，Mac → iPhone）
  - **iCloud Drive**（跨平台）
  - **邮件附件**（任何场景）
- [ ] iPhone 打开 AltStore → `My Apps` tab → `+` → 选择文件
- [ ] AltStore 开始签名 + 安装（首次会下载 Apple 证书）
- [ ] 进度条完成后，主屏出现 `审计官` 图标

**16:00-17:00 信任证书 + 启动**

  - [ ] 找到开发者应用（你的 Apple ID 邮箱）
  - [ ] 点击 → `信任 "Apple Development: your@email.com"`
