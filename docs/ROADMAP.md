### 验收标准（v1.0）

| 维度 | 目标 |
|---|---|
| 功能 | 20 项 P0 全部完成 + 用户验收 |
| 性能 | 冷启动 ≤ 2s · 主页 ≤ 1s · 滚动 ≥ 55 FPS |
| 质量 | 测试覆盖率 ≥ 80%（关键模块 100%）|
| 安全 | SQLCipher 加密 + Face ID |
| 兼容 | iOS 16.0+ · iPhone SE ~ 15 Pro Max |
| 稳定 | 连续使用 7 天无崩溃 |

### 不做（v1.0 明确排除）

- ❌ 圆桌统计（多成员家庭）
- ❌ 短信/邮件/通知栏自动识别
- ❌ iCloud 多设备同步
- ❌ Android 平台
- ❌ OCR / Siri Shortcuts（P1 才做）

## 🔄 路线调整原则

1. **用户价值优先**：每个版本必须给用户带来明确可感知价值
2. **架构稳定**：v1.0 完成后，v1.x 只做功能增强，不破坏架构
3. **数据可迁移**：新版本必须兼容旧版本数据
4. **技术债可见**：每个版本留 10-20% 时间清理技术债
5. **AI 渐进**：v1.0 攒攒基础 → v1.1 OCR/NLP → v2.0 深度 AI

---

## ❌ 明确不做（永久排除）

- ❌ **商业化**：不做付费 / 订阅 / 广告
- ❌ **多用户**：不做家庭共享 / 团队记账
- ❌ **第三方集成**：不做银行 API / 微信账单导入 / 短信识别
- ❌ **桌面端**：只做移动端
- ❌ **Web 版**：只做移动端
- ❌ **开源**：自用项目，不开源不发布
